The 'lab' directory is the directory defining your laboratory.

It includes at least:
	1) a 'config' sub-directory where you store as many lab configuration files as you wish.
	2) an 'intruments' sub-directory that contains
		- all the files that define your instruments.
		- some generic frontpanels provided with Quantrolab.
